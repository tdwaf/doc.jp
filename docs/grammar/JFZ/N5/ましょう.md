# ましょう Verb Ending

#### The ましょう verb ending 

The `ましょう verb ending` is added to the verb root to say things like `let's go`, or `let's eat`. In question form, ましょう means `Shall we~?`

!!! example "Examples"
    - `いきましょう` → `Let's go`
    - `帰りましょう` → `Let's return`
    - `たべましょう` → `Let's eat`
